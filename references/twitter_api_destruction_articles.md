# Twitter/X API Destruction Under Elon Musk - Reference Articles

## Timeline
- Jan 12, 2023: Third-party clients (Tweetbot, Twitterrific) silently cut off
- Jan 19, 2023: Twitter confirms ban was intentional
- Feb 1, 2023: Free API end announced (9 days notice)
- Feb 9, 2023: Free API shut off
- Mar 29, 2023: New tiers announced: Free (useless), Basic ($100/mo), Enterprise ($42,000/mo)
- Apr 2023: Emergency service bots lose access, partial reversal forced
- Oct 2024: Basic tier doubled to $200/mo
