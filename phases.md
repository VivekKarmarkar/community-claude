# Phases

## Phase 0: Twitter/X Scraper on Laptop
Build the application that scrapes Twitter/X feeds of Claude Code team members, deduplicates, and extracts signal from noise using the Claude Agent SDK. Output is a file we can look at. Runs locally.

## Phase 1: Scraper on the Cloud
Phase 0 but hosted in the cloud, running on a schedule.

## Phase 2: Discourse Server Setup
Set up a Discourse instance for the Claude Code community platform (jobs board, issues/fixes/hacks, gallery categories).

## Phase 3: Integration via Discourse MCP
Connect the scraper application to the Discourse MCP server so curated content is posted automatically.
